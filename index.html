Aerospace Engineering Computational Toolkit with AI Integration:

Developed a Python application to calculate thrust using data from Excel files. The program reads parameters like mass flow rate, velocity, exit pressure, static pressure, and nozzle area, computes thrust with a physics-based formula, and visualizes results using Matplotlib. It ensures accurate calculations by processing each row of data and generating a line plot to show thrust trends, simplifying thrust analysis for aerospace applications.

MATLAB-Image Processing (Archimedes)

Utilized MATLAB and image processing techniques to analyze the hot fire test of Rocket Lab's Archimedes rocket engine. Processed images by applying grayscale optimization and segmentation, employing convex hull algorithms to calculate the area of shock diamonds. This analysis provided critical insights into nozzle performance, fuel combustion efficiency, and exhaust flow characteristics, contributing to the evaluation and refinement of the engine's design and operation.
